# Speaklio

Speaklio is a mobile-first personal care dashboard prototype. It shows how a voice-first AI assistant could turn natural language into structured updates across modular plugins.

## Preview

Open `index.html` directly or run a local static server:

```powershell
node server.js
```

Then visit `http://localhost:4173`.

## Mock features

- Responsive daily dashboard for nutrition, finance, sleep, and workouts
- Plugin store with add and remove interactions
- Activity timeline with plugin filters
- Simulated assistant requests and voice input
- Phone-friendly bottom navigation and assistant sheet
